# ChatIT
Chat Application

Setup instructions :
```
git clone https://github.com/virresh/ChatIT/
cd ChatIT/
python app.py
```

Now visit localhost:8080 
The app will be live there.

Currently deployed at https://chatitflask.herokuapp.com/
